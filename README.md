AI-Powered Intrusion Detection & Prevention System (IDPS)

This project implements an AI/ML-based Intrusion Detection and Prevention System designed to detect and respond to malicious network activities in real time. Using the CSE-CIC-IDS2018 dataset, the system analyzes network traffic patterns, identifies potential threats, and classifies attacks with high accuracy. The platform supports anomaly detection, threat intelligence integration, and automated mitigation to strengthen organizational cybersecurity defenses against evolving attack vectors.

Key Features

Real-time network traffic analysis
Machine learning-based attack classification
Anomaly and behavioral threat detection
Automated intrusion prevention
Support for multiple attack categories from CSE-CIC-IDS2018
Scalable architecture for personal, startup, and enterprise deployments
